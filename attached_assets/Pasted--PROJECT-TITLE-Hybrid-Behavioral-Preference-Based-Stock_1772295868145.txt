🧠 PROJECT TITLE

Hybrid Behavioral + Preference-Based Stock Recommendation & Portfolio Allocation Engine

🎯 PROJECT OBJECTIVE

Build a full-stack web application that:

Authenticates users

Stores multiple investment profiles per user

Learns behavioral preference from stock selections

Collects explicit investment preferences

Combines both into a hybrid weight vector

Scores stocks

Allocates capital intelligently

Returns portfolio recommendation

Allows profile reuse and history tracking

🏗 SYSTEM ARCHITECTURE
Quant Metrics Engine
        ↓
Django Backend (REST API)
        ↓
PostgreSQL Database
        ↓
Recommendation Engine
        ↓
Portfolio Allocation Engine
        ↓
React Frontend

🟢 PHASE 6 — BEHAVIORAL ENGINE
Step 6.1 — Return Random Stocks

Endpoint:

GET /api/stocks/random

Return 10 random stocks with:

ticker

sector

normalized metrics

price

Step 6.2 — Submit Behavioral Response

POST /api/behavioral/submit

Input:

profile_name

liked_stock_ids

Backend logic:

Create new InvestmentProfile

Store BehavioralInteraction entries

Compute Behavioral Vector

Step 6.3 — Compute Behavioral Vector

For liked stocks:

For each feature:

mean(norm_feature_i)

Normalize vector so sum = 1.

Store as behavioral_vector.

🟢 PHASE 7 — PREFERENCE ENGINE
Step 7.1 — Submit Preferences

POST /api/preferences/submit

Input:

investment_goal

preferred_sectors

geography

capital

Step 7.2 — Convert Goal → Preference Vector

Example mapping:

Capital Preservation:

recovery high

volatility low

Wealth Growth:

cagr high

alpha high

Short-term:

volatility high

recovery low

Normalize P so sum = 1.

Store in profile.

🟢 PHASE 8 — HYBRID WEIGHT COMPUTATION

Inside recommendations service:

FinalWeight = alpha * B + (1 - alpha) * P

Where:

alpha = 0.7 (default)

Store final_weight_vector.

🟢 PHASE 9 — RECOMMENDATION ENGINE

Endpoint:

POST /api/recommend

Input:

profile_id

mode (single or diversify)

Step 9.1 — Load Profile

Verify profile belongs to authenticated user.

Load:

final_weight_vector

capital

sector filter

geography filter

Step 9.2 — Score All Stocks

For each stock:

Score = Σ(weight_i × norm_feature_i)
Step 9.3 — Apply Filters

Filter by:

preferred_sectors

geography

Step 9.4 — Rank

Sort by Score descending.

Select top 5.

🟢 PHASE 10 — PORTFOLIO ALLOCATION ENGINE
If Mode = Single

Select highest score stock.

Compute:

max_shares = capital // price

Return recommendation.

If Mode = Diversify

Compute:

Allocation_i = Score_i / sum(Score_top5)
Investment_i = Allocation_i × capital
Shares_i = floor(Investment_i / price_i)

Return structured portfolio plan.
